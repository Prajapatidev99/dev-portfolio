# Dev Prajapati — Portfolio

Personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — lightning-fast dev server & build
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations
- **Lucide React** — icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
